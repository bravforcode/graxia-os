# Incident Response Runbook

## Roles

- incident commander
- communications owner
- technical lead

## First 15 Minutes

1. confirm blast radius
2. freeze risky changes
3. capture timestamps and symptoms
4. verify health/readiness endpoints
5. decide rollback vs forward-fix

## Evidence

- failing endpoint/path
- latest deployment or commit
- logs/metrics summary without secrets

## Exit Criteria

- customer-facing issue mitigated
- rollback or fix verified
- follow-up owner assigned
