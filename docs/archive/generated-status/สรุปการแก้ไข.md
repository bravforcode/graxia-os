## 🚀 วิธี Deploy

### ขั้นตอนสั้นๆ:
```bash
# 1. ตั้งค่า environment
cp .env.example .env
# แก้ไข .env ใส่ API keys จริง

# 2. Start services
docker-compose up -d postgres redis

# 3. Run migrations
cd backend
alembic upgrade head

# 4. Start backend
uvicorn app.main:app --reload

# 5. Start frontend
cd frontend
bun run dev

# 6. เปิด browser
# http://localhost:3000
```
