### 4. ✅ Scorer Agent - ใช้ Domain Events แล้ว

**ไฟล์:** `backend/app/agents/scorer.py`

**เปลี่ยนจาก:**
```python
# ❌ แบบเดิม (dict)

})
```

**เป็น:**
```python
# ✅ แบบใหม่ (Domain Event + Value Object)
from app.core.domain_events import OpportunityScored
from app.core.value_objects import Score

event = OpportunityScored(
    opportunity_id=str(opp_id),
    score=Score(85.5),
    reasoning="High match",
    action_priority="do_now"
)
await self.bus.emit_domain_event(event)
```

**ผลลัพธ์:**
- ✅ Type-safe
- ✅ ใช้ Value Objects
- ✅ ใช้ Domain Events

### 5. ✅ Event Bus - รองรับ Domain Events

**ไฟล์:** `backend/app/core/event_bus.py`

**เพิ่ม Method ใหม่:**
```python
async def emit_domain_event(self, domain_event: DomainEvent) -> None:
    """Emit typed Domain Event"""
    payload = domain_event.to_dict()
    await self.emit(domain_event.event_name, payload)
```

**ผลลัพธ์:**
- ✅ รองรับ Domain Events
- ✅ Type checking
- ✅ Backward compatible
