## 🔌 API Endpoints (50+ Endpoints)

### Agent Ecosystem APIs
```
POST   /api/v1/agents                          # Create agent
GET    /api/v1/agents                          # List agents
GET    /api/v1/agents/{id}                     # Get agent
PUT    /api/v1/agents/{id}                     # Update agent
DELETE /api/v1/agents/{id}                     # Delete agent

POST   /api/v1/agents/{id}/skills              # Assign skill
DELETE /api/v1/agents/{id}/skills/{skill_id}   # Remove skill
GET    /api/v1/agents/{id}/skills              # List agent skills

POST   /api/v1/agents/{id}/teams               # Join team
DELETE /api/v1/agents/{id}/teams/{team_id}     # Leave team

GET    /api/v1/agents/{id}/reputation          # Get reputation
POST   /api/v1/agents/{id}/feedback            # Submit feedback

POST   /api/v1/agents/collaborate              # Start collaboration
GET    /api/v1/agents/marketplace               # Browse marketplace
POST   /api/v1/agents/marketplace/buy          # Purchase skill

GET    /api/v1/agents/leaderboard              # View leaderboard
GET    /api/v1/agents/{id}/certificates        # Get certificates
POST   /api/v1/agents/{id}/wishlist            # Add to wishlist
```

### Workflow & Automation APIs
```
POST   /api/v1/workflows                       # Create workflow
GET    /api/v1/workflows                      # List workflows
GET    /api/v1/workflows/{id}                # Get workflow
PUT    /api/v1/workflows/{id}                # Update workflow
DELETE /api/v1/workflows/{id}                # Delete workflow

POST   /api/v1/workflows/{id}/execute         # Execute workflow
GET    /api/v1/workflows/{id}/executions      # List executions
POST   /api/v1/workflows/{id}/trigger          # Set trigger

POST   /api/v1/pipelines                      # Create pipeline
GET    /api/v1/pipelines/{id}/nodes            # Get nodes
POST   /api/v1/pipelines/{id}/nodes           # Add node
PUT    /api/v1/pipelines/{id}/nodes/{node_id}  # Update node

POST   /api/v1/automation/schedule            # Schedule task
DELETE /api/v1/automation/schedule/{id}       # Cancel schedule
GET    /api/v1/automation/events                # Get events
```
