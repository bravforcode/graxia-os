# 60-Day Paper Trade Validation Checklist

## Prerequisites
- [ ] All Phase A tasks completed
- [ ] All Phase B tasks completed
- [ ] All Phase C tasks completed
- [ ] System running for 60 consecutive days

## Daily Checks (Automated)
- [ ] System uptime > 99%
- [ ] No critical errors in logs
- [ ] All safety gates functioning
- [ ] Kill switch tested weekly

## Weekly Checks (Manual)
- [ ] Review trade decisions against market
- [ ] Check risk limits not exceeded
- [ ] Verify reconciliation results
- [ ] Review Telegram notifications

## Performance Metrics
- [ ] Win rate > 45%
- [ ] Profit factor > 1.2
- [ ] Max drawdown < 10%
- [ ] Sharpe ratio > 1.0

## Safety Checks
- [ ] Kill switch tested (trigger and reset)
- [ ] Circuit breaker tested
- [ ] News blackout gate tested
- [ ] Correlation check tested

## Go/No-Go Criteria
- [ ] All daily checks pass for 60 days
- [ ] All weekly checks pass
- [ ] Performance metrics meet thresholds
- [ ] Safety checks all pass
- [ ] Human approval of live mode switch
