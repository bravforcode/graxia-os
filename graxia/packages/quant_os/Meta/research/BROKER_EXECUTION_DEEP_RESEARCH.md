## Key Technical Takeaways for Gracia/Ruflow

### Pepperstone Razor Account (Recommended for XAUUSD)
- **Spread:** Raw from 0.0 pips on EUR/USD; XAUUSD typically 0.2-0.4 pips
- **Commission:** $7/lot round-turn (commodities including gold)
- **Execution:** ECN model, Equinix LD4 (London) servers
- **Regulation:** ASIC (primary), FCA, CySEC, SCB, DFSA, CMA
- **API Access:** MT4, MT5, cTrader — all support Python via respective libraries

### MT5 Python Integration Architecture
```
pip install MetaTrader5    # v5.0.5735, MIT License
import MetaTrader5 as mt5

# Connection
mt5.initialize()
mt5.login(account=..., password=..., server=...)

# Order Flow
request = {
    "action": mt5.TRADE_ACTION_DEAL,
    "symbol": "XAUUSD",
    "volume": 0.01,
    "type": mt5.ORDER_TYPE_BUY,
    "price": mt5.symbol_info_tick("XAUUSD").ask,
    "deviation": 20,
    "magic": 123456,
    "comment": "gracia_paper",
    "type_time": mt5.ORDER_TIME_GTC,
    "type_filling": mt5.ORDER_FILLING_IOC,
}
result = mt5.order_send(request)
```

### Critical Integration Points
1. **Connection Resilience:** Wrap all MT5 calls in try/except with `mt5.last_error()` checking and automatic reconnect
2. **Paper → Shadow → Live Pipeline:** Use MT5 demo first, then shadow mode (parallel execution comparison), then live
3. **Kill Switch:** Implement `mt5.TRADE_ACTION_REMOVE` for pending orders + `TRADE_ACTION_DEAL` with `ORDER_TYPE_SELL` for immediate close
4. **Slippage Control:** Set `deviation` parameter in `MqlTradeRequest`; monitor with `fxblue.com` tools
5. **Reconciliation:** Compare MT5 `history_deals_get()` against strategy logs every trading session
