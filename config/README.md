# Configuration Files

This directory (`config/`) contains configuration files for the various components of the Graxia OS stack:

- **Docker Configs:** Container configurations, environment definitions, and compose overlays.
- **PM2 Configs:** Process management configurations for backend/frontend services (e.g., `ecosystem.config.js`).
- **Redis Configs:** Redis server configurations (if applicable).
- **Testing Configs:** Configurations for pytest, vitest, or other testing frameworks.

Having these consolidated here ensures that the root directory remains clean and configurations are easy to discover and manage.
