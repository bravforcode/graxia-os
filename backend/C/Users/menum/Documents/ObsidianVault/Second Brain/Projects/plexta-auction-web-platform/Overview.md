# Plexta — Auction Web Platform

## Tagline
Real-time web auction platform with live bidding and seller/buyer management

## Description
Plexta is a full-stack auction web platform that handles real-time bidding, item listings, user authentication, and transaction management. Built for the growing Thai online auction and e-commerce resale market — reliable experience for both sellers and buyers, no third-party auction fees.


## Tech Stack
