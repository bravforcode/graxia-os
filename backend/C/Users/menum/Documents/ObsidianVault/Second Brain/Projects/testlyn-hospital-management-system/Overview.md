# Testlyn — Hospital Management System

## Tagline
Full-stack hospital management system that digitizes clinical and operational workflows

## Description
Many clinics and small hospitals in Thailand still run on paper or basic spreadsheets. Testlyn is a hospital management system that digitizes patient records, appointment scheduling, and staff workflows into one reliable platform — reducing manual errors and freeing up staff time for patient care.
